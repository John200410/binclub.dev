BINSCURE

A commercial Java obfuscator
Secure

Binscure protects your java applications from reverse engineering, decompilation, and pirating. We use advanced and unique techniques to hinder attackers attempting to reverse your application such as cryptographic string encryption, control flow obfuscation and constant value obfuscation.

Easy to use

Binscure is easy to use. Simply run it passing the configuration file as an argument and it will produce an output jar in no time. Its configuration file allows scripting like behaviour allowing you to suit your exact development environment.

Peace of Mind

Obfuscating your program with Binscure gives you peace of mind: You can spend your time making a great product, we will handle the security.

Fast

Binscure is fast to setup, fast to run. Binscure requires you to provide minimal runtime libraries, and the configuration format is fully documented with examples at github.com/binclub/BinscureDocumentation.
